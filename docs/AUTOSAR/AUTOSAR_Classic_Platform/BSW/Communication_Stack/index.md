# Communication Stack

- Com - Signal Communication Manager
- PduR - PDU Router
- CanIf - CAN Interface
- CanTp - CAN Transport Protocol
- LinIf - LIN Interface
- LinTp - LIN Transport Protocol
- FrIf - FlexRay Interface
- EthIf - Ethernet Interface
- SomeIp - SOME/IP (Adaptive)
